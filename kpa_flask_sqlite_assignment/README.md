# KPA API Assignment (SQLite Version)

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the app:
   ```bash
   python app.py
   ```

## API Endpoints

- `POST /api/kpa-form`
- `GET /api/kpa-forms`

This version uses SQLite instead of PostgreSQL. A file `kpa.db` will be created locally.
